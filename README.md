# React-TaskManager
## Task manager

## Installation

Use your console to install React-TaskManager.

```bash
gitclone https://github.com/NotNikita/React-TaskManager.git
```

## Usage

First install dependencies run JSON-server:
```javascript
npm install
json-server --watch db.json --port 5000
```

then run dev:
```javascript
npm start
```
or production:
```javascript
npm i -g serve
serve -s build -p 8000
```
